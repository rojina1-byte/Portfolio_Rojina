About

This weather app provides real-time temperature, humidity, and forecast data based on your location. It is designed to be simple, fast, and user‑friendly, offering accurate weather updates at a glance. Whether you're planning your day or checking upcoming conditions, this app has you covered.

Project

1. Real-Time Weather Data

Shows live temperature, humidity, wind speed, and weather conditions using data fetched instantly from a weather API.

2. Location-Based Updates

Automatically detects the user’s location or allows manual city input to give accurate, local weather information.

3. Clean & Simple UI

A minimal, easy-to-read interface optimized for fast access and works smoothly on both mobile and desktop devices.

4. Forecast Information

Includes hourly or weekly forecasts with weather icons for clearer and more visual weather updates.
